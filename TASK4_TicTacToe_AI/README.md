# 🎮 Tic-Tac-Toe AI

A Python-based Tic-Tac-Toe game where a human player competes against an AI. The AI uses the **Minimax Algorithm** to make optimal moves, making it impossible to beat.

## Features

- Human vs AI gameplay
- AI powered by the Minimax algorithm
- Winner and draw detection
- Input validation
- Play Again option
- Simple command-line interface

## Technologies Used

- Python
- Minimax Algorithm

## How to Run

1. Clone the repository.
2. Navigate to the project folder.
3. Run the following command:

```bash
python tictactoe.py
```
