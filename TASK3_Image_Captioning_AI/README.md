# 🖼️ AI Image Captioning

An AI-powered web application that generates captions for uploaded images using the BLIP Image Captioning model.

## Features

- Upload JPG, JPEG, and PNG images
- Generate AI-based image captions
- View uploaded image
- Download generated caption
- Caption history
- Simple and interactive UI

## Technologies Used

- Python
- Streamlit
- Hugging Face Transformers
- PyTorch
- Pillow

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
streamlit run app.py
```
