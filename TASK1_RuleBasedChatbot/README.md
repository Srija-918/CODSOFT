🤖 RuleBot - Rule-Based AI Chatbot

A professional rule-based chatbot developed using Python for the CodSoft Artificial Intelligence Internship.

📌 Overview

RuleBot is a simple conversational AI system that responds to user inputs using predefined rules and pattern matching techniques. The project demonstrates basic concepts of:

- Natural Language Processing (NLP)
- Conditional Logic
- Pattern Matching
- Conversation Flow Design
- Modular Python Programming

✨ Features

- Friendly greetings
- Answers questions about AI, Python, ML, and NLP
- Displays current date and time
- Tells jokes
- Provides motivational quotes
- Handles unknown inputs gracefully
- Typing animation for realistic interaction
- Modular code structure

🛠️ Technologies Used

- Python 3
- Built-in modules:
  - "random"
  - "time"
  - "datetime"

🚀 How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run:

python chatbot.py

💬 Sample Interaction

You: hi
Bot: Hello! 👋

You: what is ai
Bot: Artificial Intelligence enables machines to perform tasks that normally require human intelligence.

You: tell me a joke
Bot: Why do programmers hate nature? Because it has too many bugs.
