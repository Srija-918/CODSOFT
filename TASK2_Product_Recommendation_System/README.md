# 🛍️ Product Recommendation System

An AI-powered Product Recommendation System built using Python and Streamlit that helps users discover similar electronic products based on their search input.

## 🚀 Features

- 🔍 Search products instantly
- 🤖 AI-based product recommendations
- 💰 Localized pricing in Indian Rupees (₹)
- 📊 Dashboard statistics
  - Total Products
  - Average Price
  - Highest Price
- 🎯 Product discounts and rankings display
- 🖥️ Interactive and user-friendly interface

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
